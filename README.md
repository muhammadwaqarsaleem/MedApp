# MedApp

MedApp is a healthcare management system developed as part of a Software Design & Architecture (SDA) project.  
It provides a structured solution for managing patient records, appointments, and operational workflows in a clinical setting.

---

## Features
- **Patient Record Management**: Secure storage and retrieval of patient information.  
- **Appointment Scheduling**: Efficient booking and tracking of appointments.  
- **Modular Design**: Built with SDA principles to ensure scalability and maintainability.  
- **User-Friendly Interface**: Intuitive design for ease of use by healthcare staff.  
- **Documentation**: Clear design rationale and workflow explanations for onboarding and sustainability.

---

## Tech Stack
- **Language**: Python (Django Framework)
- **Frameworks/Tools**: Django
- **Architecture**: Modular design with separation of concerns, following SDA best practices.

## Project Context
This project was developed as part of an academic SDA course to demonstrate the practical application of software architecture principles.
The focus was on scalability, maintainability, and usability, ensuring the system can serve as a foundation for future healthcare solutions.
